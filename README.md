# darsHub - O'quv Markazi Boshqaruv Tizimi

Full-stack o'quv markazi boshqaruv tizimi. Frontend (Vue.js) va Backend (Node.js/Express) alohida papkalarda.

## 📁 Loyiha Struktura

```
darsHub/
├── front/                    # Frontend (Vue.js + Vite)
│   ├── src/
│   ├── package.json
│   └── ...
├── backend/                  # Backend (Node.js + Express)
│   ├── src/
│   ├── package.json
│   └── ...
└── README.md
```

## 🚀 Tez Start

### Frontend o'rnatish va ishga tushirish

```bash
cd front
npm install
npm run dev
```

Frontend: `http://localhost:5173`

### Backend o'rnatish va ishga tushirish

```bash
cd backend
npm install
npm run dev
```

Backend: `http://localhost:5000`

**Frontend:**
- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Pinia (State Management)
- Vue Router

**Backend:**
- Node.js
- Express.js
- TypeScript
- PostgreSQL (Supabase)
- JWT Authentication
- bcryptjs
